## Giới Thiệu
Đây là thư mục chứa script dành riêng cho việc mở cùng lúc nhiều trang web làm airdrop 
Có thể đối với bạn nó không hay, nhưng nó tiện với mình ^^

# Tính Năng
Chỉ có 1 tính năng duy nhất là mở 1 lần nhiều tab,Để làm các task daily & tránh bị quên

## Sử Dụng Như Thế Nào ?
Chỉ cần click vào file .bat & chạy
Nếu muốn thêm,sửa hay xoá website nào thì edit file .bat bằng notepad  và chỉnh sửa lại trang web bạn muốn là được
